## Project Overview

SmartMaintain is a hybrid machine learning system for predictive maintenance 
in manufacturing. It combines Long Short-Term Memory (LSTM) networks for 
temporal anomaly detection and Remaining Useful Life (RUL) estimation with 
Random Forest for fault classification.

## Datasets Used

| Dataset | Source | Purpose |
|---|---|---|
| NASA C-MAPSS Turbofan Engine | Kaggle / NASA | Primary — RUL estimation |
| Microsoft Azure PdM | Kaggle | Secondary — Fault classification |
| CWRU Bearing Fault Dataset | Case Western Reserve University | Validation |
